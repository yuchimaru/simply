# Simpleな汎用自己紹介用テンプレート一式

bootstrapでグリッドを使ったシングルページです。

## ■使用方法

必要に応じてindex.html/style.cssを編集して、
自分向けのページに作成ください。
画像類はご自身のファイルをご利用ください。

## ■必要な知識

html/css/javascirpt

cssフレームワーク
bootstrap
